# 🛡️ FortiX-Conky

Conky **ligero y elegante** diseñado especialmente para **FortiX Linux**.  
Monitorea en tiempo real tu equipo sin consumir recursos.

---

## 📊 ¿Qué muestra?

- 🧩 **CPU**
- 📈 **RAM**
- 💾 **Disco**
- 🌐 **Red (subida/bajada)**
- 🔋 **Batería**
- 🌡️ **Temperatura**
- 🕒 **Hora y fecha**
- ⚙️ Compatible con zzz-icewm
